# 📊 Predictive Analytics using Linear Regression

## 🚀 Project Overview
This project uses machine learning (Linear Regression) to predict future sales based on historical data.

## 📌 Features
- Data preprocessing
- Regression model for prediction
- Forecasting future sales
- Data visualization using graphs

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📂 Dataset
The dataset contains sales data over a period of days.

## ▶️ How to Run
1. Install libraries:
   pip install pandas matplotlib scikit-learn

2. Run:
   python prediction.py
